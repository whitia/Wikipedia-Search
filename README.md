更新履歴
======================
(2014/8/7)
・取得する要素を「項目名」「本文」「本文内のリンク項目」の3つに分け、それぞれを取得する処理を関数化。
・上記に伴い、メイン関数呼び出し時の引数に「項目名」「本文」「本文内のリンク項目」を表すプロパティを追加。
　これにより取得する要素が指定可能となった。
・本文内のリストが取得可能となった。
・本文内のリンク項目に半角スペースが含まれている場合、リンクが途切れる不具合を修正。
・ページ上部に検索した語句の履歴を表示するようにした。
